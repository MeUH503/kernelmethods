API Reference¶

A tutorial-like presentation is available at Usage.

This library consists of a set of key classes such as KernelMatrix documented in KernelMatrix class, diverse library of Kernel functions, KernelSet and KernelBucket described in Collection of kernel matrices, along with a library of Kernel Operations and related Utilities.
