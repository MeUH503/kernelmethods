API Reference¶

A tutorial-like presentation is available at Usage.

This library consists of a set of key classes and methods such as

  • diverse library of Kernel functions,
  • KernelMatrix documented in KernelMatrix class,
  • container classes KernelSet and KernelBucket described in Collections of kernel matrices,
  • a library of Kernel Operations and
  • a set of related Utilities.
